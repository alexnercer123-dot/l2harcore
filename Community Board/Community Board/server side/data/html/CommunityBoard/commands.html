Game functions
To use any function enter in chat.Game functions contains convenient capabilites.
List of functions
Function Parameters Description
.cfg - Main settings of your character
.whoiam - More information about your character
.offline - Offline trade (from 40lvl)
.repair (name) Repair of your character (if you have some problems)
.recallclan - Recall clan members (only for leaders)
.lock - Lock your account by HWID
.password - Changes your password
.premium - Check your premium account
.play - Auto-Play service menu
